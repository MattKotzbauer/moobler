# App Keybindings Reference

## Global (all screens)
| Key | Action |
|-----|--------|
| `1` | Go to Home screen |
| `2` | Go to Config screen |
| `3` | Go to Discover screen |
| `4` | Go to Sandbox screen |
| `j` / `↓` | Navigate down |
| `k` / `↑` | Navigate up |
| `h` / `l` | Switch panels (Discover), switch tabs (Config) |
| `Enter` | Select / activate |
| `q` | Quit |
| `Ctrl+C` | Quit |
| `?` | Show help |

## Home Screen (1)
| Key | Action |
|-----|--------|
| `j` / `k` | Navigate menu items |
| `Enter` | Select menu item |
| `m` | Show moobler notification |

## Config Screen (2)
| Key | Action |
|-----|--------|
| `j` / `k` | Scroll keybindings list |
| `h` / `l` / `Tab` | Switch tabs (Keybindings / Style) |

## Discover Screen (3)
| Key | Action |
|-----|--------|
| `h` | Focus categories panel (left) |
| `l` | Focus suggestions panel (right) |
| `j` / `k` | Navigate items in focused panel |
| `Enter` | Select category / try keybind |
| `s` | Search Online (fetch AI suggestions) |
| `t` | Try selected keybind in sandbox |

## Sandbox Screen (4)
| Key | Action |
|-----|--------|
| `s` | Start sandbox (launches Kitty with Docker) |
| `j` / `k` | Navigate UI elements |
