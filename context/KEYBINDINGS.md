# App Keybindings Reference

## Global (all screens)
| Key | Action |
|-----|--------|
| `1` | Go to Home screen |
| `2` | Go to Config screen |
| `3` | Go to Discover screen |
| `4` | Go to Sandbox screen |
| `j` / `k` | Navigate down / up |
| `h` / `l` | Navigate left / right (context-dependent) |
| `g` / `G` | Jump to top / bottom |
| `Ctrl+d` / `Ctrl+u` | Page down / up |
| `Enter` / `o` | Select / activate |
| `Esc` | Go back |
| `q` | Quit |
| `?` | Show help |

## Config Screen (2)
| Key | Action |
|-----|--------|
| `j` / `k` | Navigate table rows |
| `h` / `l` | Switch tabs (Keybindings / Style / Options) |
| `g` / `G` | Top / bottom of table |

## Discover Screen (3)
| Key | Action |
|-----|--------|
| `h` | Focus categories panel (left) |
| `l` | Focus suggestions panel (right) |
| `j` / `k` | Navigate items in focused panel |
| `Enter` | Select category / try keybind |
| `t` | Try selected keybind in sandbox |
| `a` | Add selected keybind to config |
| `d` | Show keybind details |

## Within KeybindCard (when focused)
| Key | Action |
|-----|--------|
| `h` / `l` | Move between Try/Add/Details buttons |
| `Enter` / `o` | Activate highlighted button |
| `t` | Quick: Try It |
| `a` | Quick: Add to config |
| `d` | Quick: Show details |

## Sandbox Screen (4)
| Key | Action |
|-----|--------|
| `s` | Start sandbox container |
| `x` | Stop sandbox container |
| `j` / `k` | Navigate buttons |
| `Ctrl+d` / `Ctrl+u` | Scroll log |
